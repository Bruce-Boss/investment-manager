# 投资管理系统

> 一个用来管理某投资 APP 个人资产的移动端优先的响应式网页 APP, 已上线并给一些朋友使用，深受群众喜爱;)

### URL

- http://invest.kzhang.site/

### 技术栈

##### 前端

- React
  - React-hooks
  - Material-UI
  - Mobx
  - React Router DOM

##### 后端

- Express
- Mongoose

##### 数据库

- MongoDB
